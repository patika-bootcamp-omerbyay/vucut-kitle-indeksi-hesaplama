# ⚖️ Vücut Kitle İndeksi (VKI) Hesaplayıcı

Bu Java programı, kullanıcının boy ve kilo bilgilerini alarak sağlık durumunun genel bir göstergesi olan Vücut Kitle İndeksini hesaplar.

## 🚀 Kullanım

Program çalıştırıldığında sizden sırasıyla şu bilgileri ister:
1. **Boy:** Metre cinsinden (Örn: 1.72)
2. **Kilo:** Kilogram cinsinden (Örn: 105)

## 🛠️ Hesaplama Mantığı

Program şu formülü temel alır:
$$VKI = \frac{Kilo}{Boy \times Boy}$$
